# Drift Infinity

2D аркадный симулятор дрифта с процедурными трассами и глубокой физикой двигателя.

## Особенности

- **Реалистичный дрифт:** физика на базе Bicycle Model с расчетом Slip Angles, Yaw Moment и переносом веса.
- **Управляемый занос:** поддержка контрруления и динамической стабилизации газом.
- **Ручная трансмиссия:** 6 передач, рабочее сцепление и механика Clutch Kick.
- **Ретро-аудио:** синтезированный звук двигателя и визг шин в реальном времени.
- **Система рекордов:** сохранение лучших результатов в браузере.

## Управление

- **WASD / Стрелки** — Движение и руль (инерционное управление)
- **Space (Пробел)** — Ручной тормоз
- **Z / Left Alt** — Сцепление (удерживайте при смене передач или для инициации дрифта)
- **Left Ctrl / Left Shift** — Повышение / Понижение передачи (только с выжатым сцеплением)
- **Drift Entry** — Разогнаться, выжать сцепление (Z), зажать ручник (Space) и вывернуть руль.
- **Visual Feedback** — Белые векторы на колесах показывают скорость их вращения.

## Технологии

- Чистый JavaScript (ES6+)
- HTML5 Canvas
- Web Audio API

## Разработка

Игра построена на модульной архитектуре с фиксированным шагом физики (60 FPS) для стабильности на любых устройствах.
