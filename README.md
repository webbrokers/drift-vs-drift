# Drift Infinity

2D аркадный симулятор дрифта с процедурными трассами и глубокой физикой двигателя.

## Особенности

- **Хардкорный дрифт:** физика основана на углах увода шин (slip angles).
- **Ручная трансмиссия:** 6 передач, рабочее сцепление и механика Clutch Kick для входа в занос.
- **Процедурные трассы:** каждая сессия — это идеально сбалансированная "восьмерка" для бесконечного дрифта.
- **Ретро-аудио:** синтезированный звук двигателя и визг шин в реальном времени.
- **Система рекордов:** сохранение лучших результатов в браузере.

## Управление

- **WASD / Стрелки** — Движение и руль
- **Space (Пробел)** — Ручной тормоз
- **Left Alt** — Сцепление (удерживайте при смене передач)
- **Left Ctrl / Left Shift** — Повышение / Понижение передачи (только с выжатым сцеплением)
- **Clutch Kick** — Разгонитесь, выжмите сцепление, дайте полный газ и резко отпустите сцепление для пинка!

## Технологии

- Чистый JavaScript (ES6+)
- HTML5 Canvas
- Web Audio API

## Разработка

Игра построена на модульной архитектуре с фиксированным шагом физики (60 FPS) для стабильности на любых устройствах.
